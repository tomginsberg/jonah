## Install
```
pip install git+https://github.com/tomginsberg/jonah.git
```
## Run
```
jonah -h
usage: jonah [-h] [-o OLD] [-n NEW] [-out OUTPUT]

options:
  -h, --help            show this help message and exit
  -o OLD, --old OLD     Path to the old spreadsheet
  -n NEW, --new NEW     Path to the new spreadsheet
  -out OUTPUT, --output OUTPUT
                        Path to the output generated spreadsheet (default diff.csv)
```
